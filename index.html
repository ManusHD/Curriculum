Proyectos destacados

Aplicación Web de Gestión de Pedidos e Inventario para CHANEL España

Aplicación para digitalizar y automatizar el control de inventario de la empresa DELIM (distribuidor de Chanel). Permite registrar movimientos de producto, gestionar puntos de venta y destinos, importar datos desde Excel y validar información en tiempo real.

El sistema centraliza la información y elimina errores humanos, mejorando la eficiencia y trazabilidad de los pedidos. Además, se diseñó para escalar, adaptarse y evolucionar según las necesidades reales de la empresa.

Desarrollada en Angular 16 y Spring Boot 17, trabajando en coordinación directa con el personal logístico para adaptar el sistema a sus necesidades reales, validando cada funcionalidad con ellos y ajustando el diseño a su forma de trabajar.

Página web informativa para el Centro de Mayores Coexate

Diseñé y desarrollé una página web sencilla pero funcional para el Centro de Mayores Coexate. El objetivo era ofrecer un espacio claro y accesible donde cualquier persona pudiera informarse sobre los servicios que ofrece el centro, conocer su historia y al equipo de profesionales que trabaja allí. La web está pensada para ser intuitiva y fácil de navegar, especialmente para personas mayores o familiares que no están tan familiarizados con la tecnología.

Página web Bobble Premier League [Proyecto personal]

Desarrollo de una página web como proyecto personal para el videojuego Bobble League de Discord, proporcionando información detallada sobre clasificaciones, partidos, clubes, récords, premios y estadísticas de jugadores. La página utiliza una hoja de Google Sheets como base de datos. Actualmente, la página está creada utilizando Angular para ofrecer un rendimiento óptimo y funcionalidades avanzadas.

Como proyecto de clase, la página se migró a React respetando las funcionalidades que tenía en Angular.
